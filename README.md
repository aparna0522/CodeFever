# CodeFever
CodeFever, is a C++ based project, dealing with the measurement(amount) of the application of various forces at various places on a given rod, beam, etc(Mechanics: System Of Forces) which has included graphs and various alluring fonts, effects and tones. \
<b>Date Of Creation: 20th March, 2017. </b>

Created a website as a part of the project using Wix Platform. \
   <b>Website URL:</b> www.codefever.wixsite.com/codefever

<h3>How to run this project?</h3>

1. Clone this repository. 
2. Open your Turbo C++ compiler. 
3. Import the code. 
4. Run the code.
